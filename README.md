# Car Fault Finder Web

MVP web app for looking up vehicle fault codes and showing:
- fault summary
- likely affected part
- part image
- approximate part location inside the vehicle
- first diagnostic checks

Current demo dataset includes:
- Porsche
- Cayenne
- 4.0T V8
- Fault code: P029900

## Development

```bash
npm install
npm run dev
```

## Test

```bash
npm test
```

## Build

```bash
npm run build
```
